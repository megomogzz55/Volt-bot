Privacy Policy

Last updated: March 2026

1. Overview

Fantasy Bot is an automated tool that publishes football and Fantasy Premier League content to Facebook pages. This policy explains how we handle data.

2. Data Collection

We do not collect, store, or share any personal user data. The app only reads publicly available football data (scores, player stats, FPL data) and posts content to Facebook pages we own.

3. Facebook API Usage

This app uses the Facebook Graph API solely to publish posts to our own Facebook pages. We do not access any user's personal profile, messages, or private data.

4. Third-Party Services

We use the following public APIs to gather football data:

  • Fantasy Premier League API (fantasy.premierleague.com)
  • ESPN public data

5. Contact

If you have any questions about this privacy policy, please contact us through our Facebook page.
